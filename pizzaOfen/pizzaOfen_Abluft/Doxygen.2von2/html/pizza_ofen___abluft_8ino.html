Detailed Description

Dieses file beinhaltet die setup() und loop() Funktion.
.

Benutzte Open Source Funktionen aus dem Arduino.h und MAX6675.h :

  • void setup() , wird einmalig aufgerufen.
  • void loop() , wird staendig abgearbeitet.
  • double MAX6675::readCelsius() , liest ueber das Serial Peripheral Interface die aktuelle Temperatur ein.

Da die Funktion : millis() auf die gleichen Timerregister zugreift,
welche wir fuer unsere PWM Ausgabe angepasst haben zaehlt sie nun
eine neu Definierte Zeit. Sie wird trotzdem benutzt um eine Verzoegerung
des einlesens des Temperatursensors und der Temperaturausgabe zu realisieren.
Da das Serial Peripheral Interface zum Auslesen des Sensors nicht unendlich
schnell arbeitet ist diese Verzoegerung zwingend notwendig.
