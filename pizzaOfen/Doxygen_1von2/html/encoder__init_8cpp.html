Detailed Description

Dieses file beinhaltet die encoderInit() Funktion.

Verwendung einer Funktion aus Open Source Quelle zum initialisieren
der Interrupt Rutinen. attachinterrupt(). Dieser Aufruf bekommt 3 Parameter mitgegeben:

  • digitalPinToInterrupt(Pin) - Gewuenschter Pin .
  • Name der ISR, sozusagen die Hinsprung Adresse.
  • Art der Detecktion, steigende/fallende Flanke.

Function Documentation

◆ encoderInit()

void encoderInit ( )

Bekanntgabe der hinsprung-Adressen und
Initialisierung der benötigten Variablen der Interrupt Sub Rutinen :

  • rechtslaufISR()
  • linkslaufISR()
  • tasterPressedISR()

Ausserdem werden für die Pins, des Encoders, speziefische konfigurationen gewählt:

  • INPUT
  • PULL-UP (Widerstand)
  • detektion der steigenden Flanke
