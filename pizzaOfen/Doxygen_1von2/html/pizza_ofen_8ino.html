Detailed Description

Dieses file beinhaltet die setup() und loop() Funktion.
.

Benutzte Open Source Funktionen aus dem Arduino.h und MAX6675.h :

  • void setup() , wird einmalig aufgerufen.
  • void loop() , wird staendig abgearbeitet.
  • double MAX6675::readCelsius() , liest ueber das Serial Peripheral Interface die aktuelle Temperatur ein.
