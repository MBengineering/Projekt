Detailed Description

Dieses file beinhaltet die tasterPressedISR() Funktion.

Interrupt Sub Rutine, extern ausgeloest durch Taster des Encoders
Sie dient zum aktivieren und deaktivieren des Heizvorganges.
Ausserdem loest er beim Umschalten in den aktiven Zustand, oder Deaktivierung,
eine Ruecksetzung der LCD Anzeige aus. Pin2 <- Taster <- GND
